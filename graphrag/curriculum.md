# GraphRAG 인터랙티브 학습

> 전통 RAG가 키워드 유사도 기반이라면, GraphRAG는 **엔티티 사이의 관계**를 함께 활용하는 차세대 검색·생성 패턴이다. 본 자료는 GraphRAG 개념부터 Neo4j/Cypher 실전, 파이프라인 구축까지 단계별로 시각화한다.

## 학습 목표

- 일반 RAG의 한계와 GraphRAG가 등장한 이유를 안다
- Knowledge Graph(노드/엣지/온톨로지) 구조를 시각으로 이해한다
- Neo4j와 Cypher 쿼리 언어로 그래프 데이터를 다룰 수 있다
- GraphRAG 파이프라인(엔티티 추출 → 관계 구축 → 쿼리)을 끝까지 그린다
- Microsoft GraphRAG, LangChain 같은 실전 도구를 어떻게 골라야 하는지 안다
